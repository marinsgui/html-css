Outras formatações

Código-fonte / Pré-formatação

Podemos formatar o texto para códigos utilizando a tag CODE. Por exemplo: O comando get.ElementbyId('teste') é escrito em liguagem JavaScript.

Também podemos deixar a formatação original utilizando a tag PRE

    
num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'O número {num} é PAR')
else:
    print(f'O número {num} é ÍMPAR')
print('Fim do programa')

Citações

Podemos fazer citações simples utilizando a tag Q.


Exemplo: Como dizia Faustão: Ô loco meu!

Podemos também fazer citações completas utilizando a tag BLOCKQUOTE

Exemplo:Segundo Mettzer (2016):
A pesquisa bibliográfica é o primeiro passo de um trabalho científico.
